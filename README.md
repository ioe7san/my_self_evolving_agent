# Self-Evolving AI Agent 🚀

هذا المشروع عبارة عن هيكل لوكيل ذكاء اصطناعي مصمم ليتطور ذاتياً من خلال تحليل أدائه وتعديل ملفات التكوين الخاصة به.

## مميزات المشروع:
- **حلقة تحسين ذاتي:** يراقب الوكيل مخرجاته ويتعلم منها.
- **تحديث تلقائي:** يقوم بتعديل ملف `config.json` بناءً على النجاحات.
- **نظام حماية:** يدعم التراجع عن التغييرات التي قد تسبب فشلاً في الأداء.

## كيفية التشغيل:
1. تأكد من تثبيت المتطلبات.
2. شغل ملف `main.py`.
